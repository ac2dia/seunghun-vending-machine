### Exception Case
- 현금 결제 케이스
  - 결제 금액 부족 (예: 1000원짜리 음료수 구매 시 500원 투입)
  - 잘못된 금액권 투입 (예: 2000원짜리 지폐 투입)
- 카드 결제 케이스
